## Формат

Каждый файл — Markdown с YAML frontmatter:
- `source` — название документа
- `edition` — редакция
- `retrieved` — дата скачивания
- `url` — источник

Статьи/пункты — H2/H3, чтобы RAG-индексатор делал осмысленные чанки.
